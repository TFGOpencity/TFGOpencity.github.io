MenuCtrl
controller in module opencity

Description

Controlador menú principal. Permite consultar las diferentes categorías existentes para la ciudad en la que se encuentra el usuario.

Methods

  • getImagen(nombreImagen)

    Obtiene la imagen que se muestra en el menú principal para cada categoría disponible. La imagen varía en función de la ciudad en la que se encuentra el usuario.

    Parameters
    ParamTypeDetails
    nombreImagenString

    Nombre de la imagen requerida

    Returns
    String

    Ruta hacia la imagen

  • go(categoria)

    Permite navegar a cada una de las categorías existentes.

    Parameters
    ParamTypeDetails
    categoriaString

    Estado al que se navega

  • goTaxis()

    Permite acceder a la sección que permite consultar las paradas de taxi cercanas.

  • obtenerLocalizacion()

    Permite volver a obtener la localización de un usuario. Se emplea cuando no puede obtenerse la ubicación en el momento de iniciar la aplicación.

  • show(categoria)

    Comprueba si se muestra una categoría determinada en el menú principal. Las categorías que se muestran dependen de la ciudad en la que se ubica el usuario.

    Parameters
    ParamTypeDetails
    categoriaString

    Categoría

    Returns
    boolean

    Valor booleano que indica si debe mostrarse la categoría

Properties

  • mostrarOpciones

    Indica si deben mostrarse las categorías existentes en la ciudad del usuario.

  • mostrarReintentar

    Indica si debe mostrarse la opción de reintentar el cálculo de la ubicación.

  • noCiudad

    Indica si debe mostrarse el mensaje de ciudad no soportada.
