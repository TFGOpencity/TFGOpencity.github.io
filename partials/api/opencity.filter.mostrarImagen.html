mostrarImagen
filter in module opencity

Description

Filtro que evalúa si mostrar la imagen del elemento o una establecida por defecto. Si el usuario ha configurado que no desea que se muestren imágenes, con el objetivo de optimizar el uso de datos, se retornará el icono por defecto; en caso contrario la ruta de la imagen del elemento.

Usage

In HTML Template Binding

{{ mostrarImagen_expression | mostrarImagen }}

In JavaScript

$filter('mostrarImagen')()
