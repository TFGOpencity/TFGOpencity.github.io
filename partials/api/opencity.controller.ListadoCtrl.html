ListadoCtrl
controller in module opencity

Description

Controlador de la vista que muestra los listados de elementos cercanos. Permite listar los alojamientos, sitios de compras, complejos deportivos, experiencias, restaurantes, monumentos, playas y paradas de autobús cercanas a la posición del usuario. También permite consultar detalles de un elemento, recargar el listado de elementos cercanos y mostrar los resultados en un mapa.

Methods

  • getLoadingCategoria()

    Genera un elemento ionicLoading que indica al usuario que se están cargando los elementos cercanos de la categoría que ha seleccionado.

  • getResultadosCategoria()

    Obtiene el listado de elementos cercanos de la categoría seleccionada por el usuario.

  • mostrarItemsMapa()

    Permite visualizar en un mapa el listado de resultados cercanos.

  • recargar()

    Permite recargar la lista de elementos cercanos con la acción 'Pull to refresh'.

  • verDetalle(idItem)

    Método que permite ver el detalle de un elemento.

    Parameters
    ParamTypeDetails
    idItemint

    Identificador del elemento.

  • factory

    Factoría de la ciudad del usuario.

Events
