Settings
service in module opencity

Description

Gestiona la configuración de la aplicación y la obtención de la ubicación.

Methods

  • calcularPosicion()

    Obtiene las coordenadas de la ubicación del usuario.

  • getCiudad(location)

    Parsea la respuesta de la llamada a la API Geocode de Google para obtener la ciudad en la que se encuentra el usuario.

    Parameters
    ParamTypeDetails
    locationObject

    Respuesta API Geocode

    Returns
    String

    Ciudad usuario

  • obtenerInformacionPosicion()

    Obtiene, a partir de la posición del usuario, el detalle del tiempo y la ciudad en la que se encuentra.

  • requestLocationFailure()

    Callback que se ejecuta si el usuario no permite acceder a la ubicación del dispositivo.

  • requestLocationOK()

    Callback que se ejecuta una vez que el usuario permite el acceso a la ubicación del dispositivo.

  • solicitarUbicacion()

    Método que verifica si la ubicación está activada en el dispositivo. Si está activada la calcula, si no, solicita al usuario activar esta característica.

Properties

  • Settings

    Objeto que almacena la configuración realizada por el usuario y su ubicación.

  • geocodeURL

    URL de la API Geocode de Google.
