ListMapaCtrl
controller in module opencity

Description

Controlador de la vista que muestra el listado de elementos cercanos ubicados en un mapa. Es posible consultar el nombre de cada resultado y acceder al detalle pulsando sobre el marcador del elemento.

Methods

  • goToItem(idItem)

    Permite acceder a la vista detalle de un elemento.

    Parameters
    ParamTypeDetails
    idItemInteger

    Identificador del elemento

Properties

  • factory

    Factoría de la ciudad del usuario.

  • items

    Conjunto de elementos cercanos que se representan en el mapa.

  • map

    Mapa en el que se representa la información.

  • ubicacionActual

    Ubicación actual del usuario.
