TaxisCtrl
controller in module opencity

Description

Controlador vista taxis. Permite listar las paradas de taxis cercanas, recargar la lista de resultados, visualizarlas en un mapa o consultar una parada específica.

Methods

  • mostrarItemsMapa()

    Permite ver el listado de paradas de taxi en un mapa.

  • recargar()

    Permite recargar la lista de paradas de taxi cercanas tras hacer 'Pull to refresh'.

  • verDetalle(idParada)

    Permite consultar el detalle de una parada de taxi.

    Parameters
    ParamTypeDetails
    idParadaint

    Identificador de la parada.

Properties

  • factory

    Factoría de la ciudad del usuario.

  • items

    Listado de paradas de taxi cercanas.

  • titulo

    Título de la vista.

Events

  • eventoRecargar

    Evento que permite recargar de forma automática la vista cuando el desplazamiento del usuario es significativo.
