DetalleItemCtrl
controller in module opencity

Description

Controlador de la vista detalle. Recupera el elemento que el usuario solicita de la lista de elementos cercanos. Permite la navegación a la vista de mapa para mostrar la ubicación del elemento, guardar el elemento que se consulta y abrir la página web del elemento.

Methods

  • guardarItem()

    Invoca al servicio que permite persistir el elemento indicado por el usuario. Se notifica si el objeto se ha podido guardar correctamente.

  • openLink()

    Abre la página web del elemento que se consulta en el navegador del dispositivo.

  • showMap()

    Accede a la vista que muestra en un mapa la posición del elemento que se consulta.

Properties

  • factory

    Factoría de la ciudad del usuario.

  • idioma

    Idioma de la aplicación.

  • itemDetalle

    Detalle del elemento.

  • showGuardar

    Variable que permite mostrar/ocultar el botón de guardar en la vista.
