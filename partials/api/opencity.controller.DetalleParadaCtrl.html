DetalleParadaCtrl
controller in module opencity

Description

Controlador de la vista que muestra el detalle de una parada. El detalle de una parada comprende el listado de autubuses que pasan por la misma y el tiempo estimado para cada uno de ellos.

Methods

  • cancelarRecarga()

    Cancela la recarga automática del detalle de la parada cuando se abandona la vista.

  • detalleLinea()

    Accede al detalle de una línea.

  • guardarParada()

    Guarda la parada que se consulta. Se notifica si el objeto se ha podido guardar correctamente o si éste ya existe.

  • isTiempo()

    Evalúa si el valor del tiempo estimado de la línea es un número.

  • recargar()

    Recarga el detalle de una parada para actualizar los tiempos de espera estimados.

Properties

  • detalleParada

    Detalle de una parada.

  • factory

    Factoría de la ciudad del usuario.

  • intervalPromise

    Intervalo que permite recargar cada 30 segundos, de forma automática, el detalle de una parada.

  • showGuardar

    Variable que permite mostrar/ocultar el botón de guardar en la vista.
